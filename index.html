AsegurApp

AsegurApp es un proyecto en donde se brinda a sus clientes la posibilidad de contratar dos tipos de seguros que brindan diferentes tipos de servicios. Va desde contratar una ambulancia o un mecanico hasta ayuda legal en accidentes automovilisticos.
Las tecnologias usadas son: Html 5,css 3, javaScript, bootstrap 5, java, spring framework con arquitectura de MVC, mysql, hibernate, JUnit y Mockito.

Express

Express es un servicio de logistica donde permite controlar todos vehiculos de una empresa llevando el control de su posicion del viaje desde que inicia hasta que finaliza a traves de geolocalización, controlando los gastos totales de todo el viaje.
Las tecnologias usadas son: Html 5,css 3, javaScript, bootstrap 5, Php con arquitectura de MVC, mysql,Mustache.

Pokedex

Proyecto personal donde se usa la tematica iconica de Pokemon.Se ofrece un listado donde se puede ver informacion detallada de todos los pokemones alimentada de una Api Pokemon, permitiendo hacer ABM de los mismos.Tambien existe control de diferentes roles de usuarios (administrador, usuario comun).
Las tecnologias usadas son: Html 5,css 3, javaScript, bootstrap 5, Php con arquitectura de MVC, mysql,Mustache.
